# Aplikasi to-do-list

## Deskripsi 
Aplikasi sederhana berbasi PHP untuk mencatat tugas harian

## Fitur
- tambah tugas
- tandai tugas selesai
- hapus tugas

## struktur folder 
- index.php
- assets/ - file css

## cara menjalankan
1. salin folder ke 'htdocs/'
2. jalankan XAMPP dan buka di 'http://localhost/todolist'
